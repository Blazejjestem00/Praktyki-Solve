# Praktyki-Solve

## Komendy Git

```text
git add .                  # Dodaj wszystkie zmiany
git commit -m "what have you done"   # Zapisz zmiany lokalnie
git push                   # Wyślij zmiany na GitHub
git pull                   # Pobierz najnowsze zmiany z GitHub
git status                 # Sprawdź aktualny status repozytorium
git fetch --all            # Pobierz informacje o zmianach z serwera (bez łączenia)
```
